# 🏏 Chennai Super Kings (CSK) Fan Website

A static fan site dedicated to the **Chennai Super Kings (CSK)** — one of the most popular teams in the **Indian Premier League (IPL)**.

🔗 **Live Demo:** [https://raghu356-dot.github.io/csk-website/#](https://raghu356-dot.github.io/csk-website/#)  its my Favourite team


---

## 📌 Project Overview

This project is a **static responsive website** showcasing information related to **CSK** including:

* Team history
* Squad details
* Match schedules
* Fan content and visuals

It is built using **HTML, CSS, and JavaScript** and deployed on **GitHub Pages**.

No backend or database is used — it’s purely front-end and ideal for showcasing web development skills.

## 📝 How to Use Locally

To run the website on your system:

1. **Clone the repo:**

   ```bash
   git clone https://github.com/Raghu356-dot/csk-website.git
   ```

2. **Open the site:**

   * Navigate to the project folder
   * Open **index.html** in any modern browser

The site will load locally without any server setup.
